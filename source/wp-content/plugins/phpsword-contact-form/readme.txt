== Description ==
[PhpSword Contact Form WordPress plugin](http://69plugins.com/free-plugins/phpsword-contact-form/ "PhpSword Contact Form WordPress plugin Link at 69plugins.com") lets you add simple and secure contact form on your WordPress website. You can display the contact form on specific page of your website. Use default administrator email to receive emails or use your own custom email address.

== Installation ==
= Search and Install from WordPress admin panel =
1. If you are logged in inside your WordPress admin panel, go to Plugins menu and click "Add New" button.
2. On Install Plugins page, under Search navigation menu, enter "PhpSword Contact Form" in the search bar and click search plugins.
3. You will find PhpSword Contact Form plugin By Pradnyankur Nikam in the list.
4. Click "Install Now" button to install the plugin or click "Details" to read more. Install now link is also available on details popup window.
5. Once installed you need to activate the plugin to use it on your website.

= Manual Installation =
1. Click "Download Version 1.0" button on plugin info page to download and save the plugin on your computer.
2. You will get "phpsword-contact-form.zip" file.
3. Login to your WordPress admin panel, go to Plugins -> Add New -> select "Upload" navigation on the page. Click the browse button, select downloaded "phpsword-contact-form.zip" file from your computer and click Install Now button. Go to plugins and activate the PhpSword Contact Form plugin from the list. OR Extract "phpsword-contact-form.zip" file and upload all the files inside "phpsword-contact-form" folder to your website's "wp-content/plugins" directory. Now login to your WordPress admin panel go to plugins and activate the PhpSword Contact Form plugin from the list.
4. Reload WordPress Admin Panel & you will get a new menu "PhpSw Contact Form" on left sidebar.
5. Click and open "PhpSw Contact Form" page. Here you can change contact form setting for your website.
